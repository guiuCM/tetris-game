# 🎮 Tetris (Vanilla JS + Vite)

A simple **Tetris clone** made with **Vanilla JavaScript** and powered by **Vite** for fast development and bundling.

## 🚀 Features
- Classic Tetris gameplay
- Piece rotation with wall kicks
- Line clearing and scoring system
- Game over detection
- Fast and lightweight (no UI frameworks)

## 📦 Installation & Running

Clone the repository and install dependencies:

```bash
git clone https://github.com/guiuCM/tetris-game.git
cd tetris-game
npm install
npm run dev
```
